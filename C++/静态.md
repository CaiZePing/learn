# 类外的 static

用 `static` 修饰的变量和函数，只能在本单元中访问，其它单元（文件）访问不到

```cpp
/**
 * @file static.cpp
 */

static int s_Variable = 5;
int Variable = 5;

static void Function()
{
	
}

```

```cpp
/**
 * @file main.cpp
 */
#include <iostream>

void Function()
{
	
}

int main()
{
	// 访问不到
	// std::cout << s_Variable << std::endl;
	// 可以访问
	std::cout << Variable << std::endl;
	// 这两个函数不会有命名冲突
	Function();
	
	return 0;
}
```

> [!CAUTION]
> 这两个文件要链接在一起，不链接将没有任何意义

# 类中的 static

用 `static` 修饰的变量和函数，在所有的实例中，这个变量和函数只有一个实例
`static` 修饰的函数中无法访问类的实例

```cpp
#include <iostream>

struct Entity
{
	static int x,y;
	// 静态函数只能访问静态变量
	static void Print()
	{
		std::cout << x << ", " << y << std::endl;
	}
};
// 需要先初始化，不然不知道在哪
int Entity::x;
int Entity::y;

int main()
{
	Entity::x = 8;
	Entity::y = 8;

	Entity::Print();
	
	Entity e;
	e.x = 5;
	e.y = 10;
	
	e.Print();
	Entity::Print();	
	
	return 0;
}
```

# 局部的 static

在一个作用域中的 `static` 修饰的变量，声明周期是一整个程序的生命周期，但是作用域没变

```cpp
#include <iostream>

// 最简单的单例实例
class Singleton
{
public:
	static Singleton& Get()
	{
		static Singleton instance;
		return instance;
	}
	
	void Hello(){}
}

int main()
{
	Singleton::Get().Hello();
	
	return 0;
}
```
