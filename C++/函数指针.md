一个指向函数的指针
这是一个在 C 中也支持的语法

# 基础语法

每个函数名，就是函数的地址

```cpp
#include <iostream>

void PrintHello(int a)
{
	std::cout << "Hello World! Value : " << a << std::endl;
}

int main()
{
	void (*func)(int) = PrintHello;
	func(10);
	func(11);
	func(12);
}
```

# 回调函数

一个函数参数列表中有一个参数是函数指针，在这个函数中使用传进来的函数

```cpp
#include <iostream>

void PrintInt(int a)
{
	std::cout << "Value : " << a << std::endl;
}

void Foreach(int *arr,size_t count,void(*func)(int))
{
	for(int i = 0;i < count;i++)
		func(arr[i]);
}

int main()
{
	int arr[5];
	arr[0] = 5;
	arr[1] = 34;
	arr[2] = 523;
	arr[3] = 78;
	arr[4] = 12;
	
	Foreach(arr,5,PrintInt);
}
```
