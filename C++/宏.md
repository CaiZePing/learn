# 简单使用

## 字面量

```cpp
#include <iostream>

#define PI 3.1415

int main()
{
	std::cout << PI << std::endl;
	return 0;
}
```

## 语句

```cpp
#include <iostream>

#define WAIT std::cin.get()

int main()
{
	WAIT;
}
```

## 内联（类似）

```cpp
#include <iostream>

#define Log(x) std::cout << x << std::endl;

int main()
{
	Log(5);
	Log("Hello");
	return 0;
}
```

## 调试

```cpp
#include <iostream>

#if DEBUG == 1
#define Log(x) std::cout << x << std::endl;
#else
#define Log(x)
#endif

int main()
{
	Log(5);
	Log("hello");
	return 0;
}
```

## 抽象

```cpp
#include <iostream>

#define MAIN int main()\
{\
	std::cout << "hello" << std::endl;\
}

MAIN
```
