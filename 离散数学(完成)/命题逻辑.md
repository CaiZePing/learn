# 命题及命题的真值

命题是表达判断的陈述句
一个判断只有两种可能：正确的判断或者错误的判断
把这种“正确”或者“错误”赋予命题，就得到命题的真值

## 命题的真值

1. 命题的真值为真：
一个命题所表达的判断与客观情况一致，计作`T(True)`
2. 命题的真值为假：
一个命题所表达的判断与客观情况不一致，计作`F(False)`

## 判断一句话是否是命题有两个关键

1. 是陈述句
2. 有且只有一个真值

## 命题的种类

**原子命题**(简单命题)

不能再分解成更简单陈述句的命题

**复合命题**(分子命题)

由若干个连结词、标点符号及原子命题复合构成的命题

## 命题的表示

用大写字母表示

# 逻辑连接词

归纳自然语言中的连结词，定义了六个逻辑连接词

## 否定 $\lnot$

表示：并非...、不...等
用于对一个命题$P$的否定，写成$\lnot P$，并读成 “非P”

### 真值表

| $P$ | $\lnot P$ |
| :-: | :-------: |
|  F  |     T     |
|  T  |     F     |

## 合取 $\land$

表示：并且、不但...而且...、既...又...、尽管...还...等

### 真值表

| $P$ | $Q$ | $P\land Q$ |
| :-: | :-: | :--------: |
|  F  |  F  |     F      |
|  F  |  T  |     F      |
|  T  |  F  |     F      |
|  T  |  T  |     T      |

## 析取 $\lor$ （或者）

可兼取的或，即两件事情可以同时发生。用 析取“$\lor$”表达

### 真值表

| $P$ | $Q$ | $P\lor Q$ |
| :-: | :-: | :-------: |
|  F  |  F  |     F     |
|  F  |  T  |     T     |
|  T  |  F  |     T     |
|  T  |  T  |     T     |

## 异或 $\veebar$ （或者）

不可兼取的或，即两件事情不能同时发生。 用异或（也称排斥或） “ $\veebar$” 表达

### 真值表

| $P$ | $Q$ | $P\veebar Q$ |
| :-: | :-: | :----------: |
|  F  |  F  |      F       |
|  F  |  T  |      T       |
|  T  |  F  |      T       |
|  T  |  T  |      F       |

## 条件 $\rightarrow$

表示：如果...那么...、若...则...等

如有 $P\rightarrow Q$，称 $P$ 是 $P\rightarrow Q$ 的前件，$Q$ 是 $P\rightarrow Q$ 的后件
也可以说 $P$ 是 $Q$ 的充分条件，$Q$ 是 $P$ 的必要条件

| $P$ | $Q$ | $P\rightarrow Q$ |
| :-: | :-: | :--------------: |
|  F  |  F  |        T         |
|  F  |  T  |        T         |
|  T  |  F  |        F         |
|  T  |  T  |        T         |

也就是说当 $P$ 为 $F$ 的时候，$Q$ 的值对最终的真值没有影响
例：$P$：土壤缺少水分; $Q$：这颗植物会死亡; $P\rightarrow Q$：如果土壤缺少水分，那么植物死亡 
在这个例子中，如果土壤缺少水分，不管这颗植物是否死亡。都不影响最终的 $P\rightarrow Q\lraa T$ 
因为土壤不缺水分的情况下，植物死没死和 $P\rightarrow Q$ 的结论不冲突

## 等价 $\lraa$

表示：当且仅当、充分必要等

| $P$ | $Q$ | $P\lraa Q$ |
| :-: | :-: | :------------------: |
|  F  |  F  |          T           |
|  F  |  T  |          F           |
|  T  |  F  |          F           |
|  T  |  T  |          T           |

$P\veebar Q$ 与 $P\lraa Q$ 的真值表有点类似

$P\veebar Q\lraa \lnot(P\lraa Q)$

## 总结

| $P$ | $Q$ | $\lnot P$ | $P\land Q$ | $P\lor Q$ | $P\veebar Q$ | $P\rightarrow Q$ | $P\lraa Q$ |
| :-: | :-: | :-------: | :--------: | :-------: | :----------: | :--------------: | :------------------: |
|  F  |  F  |     T     |     F      |     F     |      F       |        T         |          T           |
|  F  |  T  |     T     |     F      |     T     |      T       |        T         |          F           |
|  T  |  F  |     F     |     F      |     T     |      T       |        F         |          F           |
|  T  |  T  |     F     |     T      |     T     |      F       |        T         |          T           |

$\lnot$ 为一元运算
因为一个命题 $P$ 可以确定 $\lnot P$ 的真值
其余的都是二元运算
因为它们的真值必须由两个运算对象确定

# 命题符号化

命题符号化，就是将自然语言表达的句子用符号化的命题公式来表达

## 命题符号化的步骤

1. 先将语句分解成原子命题
2. 将每个原子命题用大写字母表示。注意每个原子命题都必须是一个完整的句子
3. 用确切的逻辑连接词连结原子命题，构成给定命题的符号表达式

**注意**

在考虑用什么联结词时，一定要考虑哪种逻辑连结词的真值表最符合命题描述的情况

# 命题公式及其赋值

## 命题公式

在命题公式中有三种数据类型

1. 命题常项
即命题的真值 $T \& F$
2. 常值命题
即具体命题
3. 命题变元
用大写字母表示的任一命题
命题变元本身不是命题，因为它没有固定真值，只有给它赋值，才变成命题

将一个**命题常项**或**常值命题**赋予**命题变元**的过程称为==给命题变元赋值==，也称为对命题变元作指派

## 合式公式（合法的命题公式）

1. 单个命题变元、常值命题及命题常项是合式公式
2. 若 $A$ 是合式公式，则 $\lnot A$ 是合式公式
3. 若 $A$ 和 $B$ 是合式公式，则 $(A\land B)$，$(A\lor B)$，$(A\rightarrow B)$， $(A\lraa B)$ 都是合式公式
4. 当且仅当有限次地应用⑴，⑵，⑶所得到的符号串是合式公式

**注意**

<span style="color:red">没有异或！没有异或！没有异或！</span>

**约定**

1. 最外层括号可省
2. 不影响运算次序的括号可省

**运算次序**（由高到低）

$\lnot>\land>\lor>\rightarrow>\lraa$

## 命题公式的真值表

一个含有命题变元的命题公式不是命题，因为它没有固定真值，但是给其中的所有命题变元赋值以后它就有了唯一的真值。将所有各种赋值情况汇列成表，即为该命题公式的真值表

**构造真值表的步骤**

- 由于每个命题变元都有两种赋值可能性 $(T,F)$，所以含有 $n(n ≥1)$ 个命题变元的命题公式真值表有$2^n$行
- 将 $n$ 个命题变元按字母次序排列
- 将 $F$ 记为 $0$，$T$ 记为 $1$，按照二进制数的次序赋值
- 赋值从 $00…0$ 开始，然后按二进制加法依次加 $1$， 直到 $11…1$ 为止
- 对每个赋值，计算命题公式的真值

# 命题公式的等价

**等价定义**

$A$、$B$是含有命题变元 $P_1,P_2,…, P_n$ 的命题公式，如不论给 $P_1, P_2 , …, P_n$ 何种赋值，$A$ 和 $B$ 的真值均相同，则称命题公式 $A$ 与 $B$ 等价，记作$A\lraa B$

若两个命题公式的真值表相同，则它们等价

## 基础等价公式


|  对合律   |                                      $\lnot\lnot P\lraa P$                                       |
| :----: | :----------------------------------------------------------------------------------------------: |
|  幂等律   |                              $P\lor P\lraa P$</br>$P\land P\lraa P$                              |
|  交换律   |                       $P\lor Q\lraa Q\lor P$</br>$P\land Q\lraa Q\land P$                        |
|  结合律   |        $P\lor(Q\lor R)\lraa(P\lor Q)\lor R$</br>$P\land(Q\land R)\lraa(P\land Q)\land R$         |
|  分配律   | $P\lor(Q\land R)\lraa(P\lor Q)\land(P\lor R)$</br>$P\land(Q\lor R)\lraa(P\land Q)\lor(P\land R)$ |
|  吸收律   |                      $P\lor(P\land Q)\lraa P$</br>$P\land(P\lor Q)\lraa P$                       |
| 底·摩根定律 |     $\lnot(P\lor Q)\lraa\lnot P \land\lnot Q$</br>$\lnot(P\land Q)\lraa\lnot P \lor\lnot Q$      |
|  同一律   |                              $P\lor F\lraa P$</br>$P\land T\lraa P$                              |
|   零律   |                              $P\lor T\lraa T$</br>$P\land F\lraa F$                              |
|  互补律   |                         $P\lor\lnot P\lraa T$</br>$P\land\lnot P\lraa F$                         |
|        |                                $P\rightarrow Q\lraa\lnot P\lor Q$                                |
|        |                          $P\rightarrow Q\lraa\lnot Q\rightarrow\lnot P$                          |
|        |                       $P\lra Q\lraa(P\rightarrow Q)\land(Q\rightarrow P)$                        |
|        |                         $P\lra Q\lraa(\lnot P\lor Q)\land(P\lor\lnot Q)$                         |
|        |                        $P\lra Q\lraa(P\land Q)\lor(\lnot P\land\lnot Q)$                         |

可以用真值表来证明等价

**等价公式的证明方法**

1. 列真值表
2. 用等价公式变换（用置换定律）

**置换定律**

$A$ 是一个命题公式，$X$ 是 $A$ 中的一部分且也是合式公式，如果 $X\lraa Y$，用 $Y$ 代替 $A$ 的 $X$ 得到公式 $B$，则 $A\lraa B$

## 对偶式

在一个只含有联结词 $\lnot$ 、$\lor$、$\land$ 的公式 $A$ 中， 将 $\lor$ 换成 $\land$，$\land$换成 $\lor$，$T$ 换成 $F$，$F$ 换成 $T$，其余部分不变，得到另一个公式 $A^*$，称 $A$ 与 $A^*$ 互 为对偶式

**定理**

令 $A(P_1,P_2,…,P_n)$ 是一个只含有联结词 $\lnot$、$\lor$、$\land$ 的命题公式，则 $\lnot A(P_1,P_2,…,P_n)\lraa A*(\lnot P_1,\lnot P_2,…,\lnot P_n)$

**推论**

$A(\lnot P_1,\lnot P_2,…,\lnot P_n)\lraa\lnot A*(P_1,P_2,…,P_n)$

## 等价

等价“$\lraa$”是关系符号，不是运算符号，它表明的是两个命题公式之间的关系

**等价的性质**

1. 有自反性：对任何命题公式$A$，均有 $A\lraa A$
2. 有对称性：若 $A\lraa B$，则 $B\lraa A$
3. 有传递性：若 $A\lraa B$ 且 $B\lraa C$，则 $A\lraa C$

# 重言式与矛盾式

例：

| $P$ | $\lnot P\lor P$ | $\lnot P\land P$ |
| :-: | :-------------: | :--------------: |
|  F  |        T        |        F         |
|  T  |        T        |        F         |

$\lnot P\lor P$ 为重言式（永真式）
$\lnot P\land P$ 为矛盾式（永假式）

## 定义

若公式 $A\lraa T$，则 $A$ 为重言式或永真式
若公式 $A\lraa F$，则 $A$ 为矛盾式或永假式

## 证明方式

1. 列真值表
2. 等价公式变换

## 定理

设 $A$,$B$ 为两个命题公式，$A\lraa B$当且仅当 $A\lraa B$ 是一个重言式

**证明**

若 $A\lraa B$，则不论对  $A$、$B$ 做何种赋值，$A$ 与 $B$ 的真值都相同，于是 $A\lraa B\lraa T$，即  $A\lraa B$ 是一个重言式
若 $A\lraa B$是一个重言式，则 $A\lraa B\lraa T$，于是不论 在何种赋值下，$A$ 与 $B$ 的真值均相同，即 $A\lraa B$

# 重言蕴含式

重言蕴含“$\Rightarrow$”是关系符，不是运算符

## 定义

当且仅当 $A\rightarrow B$ 是重言式，则称 $A$ 重言(永真)蕴含 $B$ ，记作 $A\Rightarrow B$
即 若 $A\rightarrow B\lraa T$，则 $A\Rightarrow B$

当左边为真的时候，右边为真

## 基本证明方法

1. 假设前件 $A$ 为真，若在此假设下能推出后件 $B$ 也为真，则 $A\Rightarrow B$ 成立
2. 假设后件 $B$ 为假，若在此假设下能推出前件 $A$ 也为假，则 $A\Rightarrow B$ 成立

## 基础重言蕴含式


| 序号  |                                 公式                                 | 序号  |                        公式                        |
| :-: | :----------------------------------------------------------------: | :-: | :----------------------------------------------: |
|  1  |                      $P\land Q\Rightarrow P$                       |  2  |             $P\land Q\Rightarrow Q$              |
|  3  |                       $P\Rightarrow P\lor Q$                       |  4  |              $Q\Rightarrow P\lor Q$              |
|  5  |                $\lnot P\Rightarrow P\rightarrow Q$                 |  6  |          $Q\Rightarrow P\rightarrow Q$           |
|  7  |                $\lnot(P\rightarrow Q)\Rightarrow P$                |  8  |    $\lnot(P\rightarrow Q)\Rightarrow\lnot Q$     |
|  9  |                     $P,Q\Rightarrow P\land Q$                      | 10  |       $\lnot P\land(P\lor Q)\Rightarrow Q$       |
| 11  |               $P\land(P\rightarrow Q)\Rightarrow Q$                | 12  | $\lnot Q\land(P\rightarrow Q)\Rightarrow\lnot P$ |
| 13  | $(P\rightarrow Q)\land(Q\rightarrow R)\Rightarrow P\rightarrow R$  |     |                                                  |
| 14  | $(P\lor Q)\land(P\rightarrow R)\land(Q\rightarrow R)\Rightarrow R$ |     |                                                  |
| 15  |      $A\rightarrow B\Rightarrow(A\lor C)\rightarrow(B\lor C)$      |     |                                                  |
| 16  |     $A\rightarrow B\Rightarrow(A\land C)\rightarrow(B\land C)$     |     |                                                  |

## 性质

1. 有自反性：对任何命题公式 $A$，有 $A\Rightarrow A$
2. 有传递性：若 $A\Rightarrow B$ 且 $B\Rightarrow C$，则 $A\Rightarrow C$
3. 有反对称性：若 $A\Rightarrow B$ 且 $B\Rightarrow A$，则 $A\lraa B$
4. 若 $A\Rightarrow B$ 且 $A\Rightarrow C$，则 $A\Rightarrow B\land C$
5. 若 $A\Rightarrow B$ 且 $C\Rightarrow B$，则 $A\lor C\Rightarrow B$

## 定理

设 $A$，$B$ 为任意两个命题公式， $A\lraa B$ 的充要条件是 $A\Rightarrow B$ 且 $B\Rightarrow A$

# 范式

## 析取范式

命题公式 $A$ 如果可等价地写成如下形式：$A_1\lor A_2\lor ...\lor A_n(n≥1)$， 其中每个项 $A_i(i=1,2,…,n)$ 是命题变元或其否定形式的合取式，称该公式为 $A$ 的析取范式

## 合取范式

命题公式 $A$ 如果可等价地写成如下形式：$A_1\land A_2\land ...\land A_n(n≥1)$， 其中每个项$A_i (i=1,2,…,n)$是命题变元或其否定形式的析取式，称该公式为 $A$ 的合取范式

## 综合

**从定义可以看出**

在析取范式与合取范式中只含有联结词 $\lnot、\land、\lor$
$\lnot$ 在命题变元之前

**析取范式与合取范式的写法**

1. 去掉 $\rightarrow$ 和 $\lraa$
2. 将 $\lnot$ 移到命题变元前，用对偶
3. 用分配律、幂等律等公式进行整理，使之成为所要求的形式

# 主析取范式

## 小项

是 $n$ 个命题变元的合取式，其中每个变元必出现且仅出现一次(以本身或否定形式)，称这个合取式为小项

- 若有 $n$ 个变元，则有 $2^n$ 个小项

每个命题的其中一个合取式，使其值为 $T$ 的，就是其中一个小项

**小项编码**

含有 $n$ 个变元的小项的角标用 $n$ 位二进制码表示
变元按字母次序排列
<span style="color:red"> 用 $1$ 表示变元本身，$0$ 表示变元的否定形式</span>

1. 每个小项当且仅当其赋值与编码相同时，其真值为 $T$
而其余 $2_{n-1}$ 组赋值均使该小项的真值为 $F$
2. 全体小项的析取式为**永真式**，记为
$\sum m_i\lraa m_0\lor m_1\lor ...m_{2^n-1}\lraa T$

## 定义

若一个命题公式的析取范式为 $A_1\lor A_2\lor ...\lor A_n(n≥1)$，其中每个 $A_i (i=1,2,...,n)$ 都是小项，则称之为该命题公式的主析取范式

## 主析取范式的求法

1.  先写出给定公式的析取范式
2. 为使每个 $A_i$ 都变成小项，对缺少变元的项 $A_i$ 要补全变元，比如缺变元 $R$，就用 “$\land(R\lor\lnot R)$”的形式补 $R$
3. 用分配律等公式加以整理

## 求主析取范式的真值表法

1. 列出给定公式的真值表
2. 找出该公式真值表中每个为“$T$”行的赋值所对应的小项
3. 用“$\lor$”联结上述小项，即可

## 定理

在真值表中，一个使公式的真值为 $T$ 的赋值所对应的小项的析取，即为此公式的主析取范式

# 主合取范式

## 大项

是 $n$ 个命题变元的析取式，其中每个变元必出现且仅出现一次(以本身或否定形式)，称该析取式为大项

- 有 $n$ 个变元，则有 $2_n$ 个大项

每个命题的其中一个析取式，使其值为 $F$ 的，就是其中一个大项

**大项的编码**

大项的编码正好与小项相反，用 $0$ 表示变元本身，$1$ 表示变元的否定形式

1. 每个大项当且仅当其赋值与编码相同时，其真值为 $F$
其余 $2_{n-1}$ 组赋值均使该大项的真值为 $T$
2. 全体大项的合取式必为**永假式**
$\prod\limits_{i=0}^{2^n-1}M_i\lraa M_0\land M_1\land...\land M_{2^n-1}\lraa F$

## 定义

若一个命题公式的合取范式为 $A_1\land A_2\land ...\land A_n(n≥1)$，其中每个 $A_i (i=1,2,…,n)$ 都是大项，则称之为该命题公式的主合取范式

## 主合取范式的求法

1. 先写出给定公式的合取范式
2. 为使每个 $A_i$ 变成大项，对缺少变元的项 $A_i$ 补全变元，比如缺变元 $R$，用“$\lor(R\land\lnot R)$”的形式补 $R$
3. 用分配律等公式加以整理

## 求主合取范式的真值表法

1. 列出给定公式的真值表
2. 找出该公式真值表中的每个为“$F$” 行的赋值所对应的大项
3. 用“$\land$”联结上述大项，即可

## 定理

在真值表中，一个使公式的真值为 $F$ 的赋值所对应的大项的合取，即为此公式的主合取范式

## 主范式

若已知其中一个主范式，如何求出另一个主范式

1. 求得一共有多少组合
$num = 2^n$ 个组合
2. 将这个主范式中的 **小项(大项)** 列下来，除了这些 **小项(大项)**，剩下的 **大项(小项)** 就是另一个主范式的组合

# 直接推理

推理就是根据一个或几个已知的判断得出一个新的判断的思维过程
称这些已知的判断为前提。得到的新判断为前提的有效结论

推理的过程就是证明永真蕴含式的过程

两个推理规则

1. P规则(引入前提规则)：在推理过程中，可以随时引入前提
2. T规则(引入结论规则)：在推理过程中，如果前面有一个或几个公式重言蕴含公式S，则可以将S纳入推理过程中

## 定义

直接推理是由一组前提，利用 P 规则、T 规则直接推演得到有效结论的方法
推理实际上就是证明永真蕴含的过程。为了使推理过程更加简单、明确，推理会采用另外一种书写格式

**书写格式**

第一列为步骤号
第二列为给定前提或得出的结论
第三列为注释列，标明是前提还是得到的结论，以及此结论是从哪几步得到的，所用公式类型

P 直接写
T(n)(m)\[I,E\] n,m 表示哪两行，I 表示是重言蕴含式，E 表示等价公式
如何分清用 E,I 你看你T后面需要几个括号，如果是一个，就是 E ，反之就是 I

# 间接推理

## 条件论证

如果要证明的结论是 $R\rightarrow S$ 的形式，则可 以把结论中 $R\rightarrow S$ 的前件 $R$ 作为附加前提，与 给定的前提一起推出后件 $S$ 即可

**定理**

如果 $H_1\land H_2\land ...\land H_n\land R\Rightarrow S$ 
则 $H_1\land H_2\land...\land H_n\Rightarrow R\rightarrow S$

**我们把上述定理写成如下规则**

CP规则 ( Conditional Proof )： 
如果 $H_1\land H_2\land ...\land H_n\land R\Rightarrow S$ 
则 $H_1\land H_2\land...\land H_n\Rightarrow R\rightarrow S$


## 反证法

**相容定义**

设 $H_1,H_2,...,H_n$ 是命题公式，$P_1,P_2,...,P_m$ 是公式中的命题变元。如果对 $P_1,P_2,...,P_m$ ，至少有一组赋值使得 $H_1\land H_2\land...\land H_n$ 的真值为 $T$，则称公式集合$\{H_1,H_2,…,H_n\}$ 是相容的(也称是一致的)

**不相容定义**

如果对 $P_1,P_2,...,P_m$ 的每一组赋值， 都使得 $H_1\land H_2\land...\land H_n$ 的真值为 $F$，则称公式集合 $\{H_1,H_2,…,H_n\}$ 是不相容的 (也称是不一致的）

### 定义

若要证明 $H_1,H_2,...,H_n\Rightarrow C$，只要证明 $\{H_1,H_2,…,H_n,\lnot C\}$ 是不相容的即可

即 若要证明 $H_1,H_2,...,H_n\Rightarrow C$，只要证明 $\{H_1\land H_2\land…\land H_n\land\lnot C\}$ 是个矛盾式即可

$(P\lor Q)\land(P\rightarrow R)\land(Q\rightarrow R)\Rightarrow R$
